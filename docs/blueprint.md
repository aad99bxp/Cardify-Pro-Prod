# **App Name**: Cardify Pro

## Core Features:

- Image Upload: Upload background images (front and back).
- Real-time Preview & Adjustment: Display the front and back background images, fields, student photo, parents' photo, and QR code placement with adjustable sizing, position for preview.
- CSV Upload: Upload CSV file with data for card generation.
- Progress Bar: Display a progress bar during the card generation process using an LLM tool to choose when to show activity, stall or stop
- Card Generation: Generate ID cards based on CSV data and background images.
- Automatic Download: Automatically download two ZIP archives containing generated ID cards, front images in one ZIP and back images in another.

## Style Guidelines:

- Primary color: Dark cyan (#3498DB) to give a modern, trustworthy feel. (HSL: 197, 61, 53)
- Background color: Very light cyan (#EBF5FB), almost white, with a hint of primary color. (HSL: 197, 44, 95)
- Accent color: Deep purple (#8E44AD) to provide contrast and highlight important interactive elements (e.g. upload buttons). (HSL: 282, 46, 45)
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text. Note: currently only Google Fonts are supported.
- Use clear, professional icons from a consistent set (e.g., Material Design Icons) for file upload, settings, and download actions.
- Clean, grid-based layout for image previews and adjustment settings. Use clear section headings and consistent spacing.
- Subtle transition animations for loading states and process completion, avoiding any abrupt changes to the UI.
